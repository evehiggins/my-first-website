About Me

Hello, I'm Eve!

I'm a student at the University of Toronto passionate about web development and creating beautiful, functional digital experiences. Now I'm taking a journey into mastering web design and development.

My Journey

Dev Diaries is a space where I document my process, challenges, and reflections while building a website from scratch using HTML and CSS. This site serves as both a learning journal and a sandbox—capturing how ideas evolve from early sketches to functional, polished web experiences. Throughout this course, I’m exploring not only the technical foundations of web development, but also how design, usability, and interactivity shape the way information is communicated online. I’m learning to use tools like Figma for planning and visual design, while becoming more confident writing clean, organized HTML and CSS. My goal is to develop the skills needed to build a well-designed, interactive website that will ultimately host my Master’s Research Project (MRP), bringing together storytelling, structure, and accessibility in a meaningful way.

Let's Connect!
